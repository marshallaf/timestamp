Timestamp API

A simple API to retrieve the requested date in Unix time and natural English.

To use, append the desired date, in either Unix or natural format, after the base url. The response will be a JSON string containing the requested date in both Unix and natural format.

example usage
// request
https://timestamp-marshallaf.herokuapp.com/145843200
https://timestamp-marshallaf.herokuapp.com/August%2016,%201974

// response
{
unix: 145843200,
natural: "August 16, 1974"
}
